How can you use the commands git log and git diff to view the history of files?

    Fill in your answer here
Команда git log позволяет узнать дату и время всех совершенных изменений, их id, краткое описание каждого изменения. 
Команда git diff позволяет сравнить любые из совершенных изменений между собой по их id и узнать конкретную разницу между данными изменениями.

The git log command allows you to find the date and time of all the changes made, to find their id, to find a brief description of each change.
The git diff command allows you to compare any of the changes made between them by their id and find out the specific difference between these changes.
